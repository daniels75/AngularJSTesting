before running Karma you have to install:
# I'm assuming here that Karma is in the 0.10 version

1. junit-reporter: npm install -g karma-junit-reporter
2. ng-scenario: npm install -g karma-ng-scenario 
# this is if you whant to run end-to-end tests form AngularJS Tutorial 

3. Install:
npm install -g karma
npm install -g karma-junit-reporter
npm install -g karma-ng-scenario 
npm install -g karma-ng-html2js-preprocessor
npm install -g karma-coverage
